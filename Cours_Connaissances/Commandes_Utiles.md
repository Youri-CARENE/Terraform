# Commandes Utiles Terraform

Voici une liste des commandes les plus courantes et leurs usages :

- **Initialisation** : `terraform init` initialise un projet.
- **Validation** : `terraform validate` vérifie la validité de la configuration.
- **Planification** : `terraform plan` prévisualise les modifications.
- **Application** : `terraform apply` applique les changements planifiés.
- **Suppression** : `terraform destroy` supprime l'infrastructure.
- **Gestion de l'État** :
  - `terraform state list` : Liste des ressources gérées.
  - `terraform state show <ressource>` : Affiche les détails d'une ressource.
- **Debugging** : `terraform refresh`, `terraform graph` pour visualiser et synchroniser l'état.
